# KAMSTRUPd


# Installing

```
sudo su -
cd /path/to/where/you/want/store/kamstrupd
git clone https://github.com/Mausy5043/kamstrupd.git
cd kamstrupd
./install.sh
./update.sh
```

# Requirements
## Hardware:
The python scripts have been shown to work correctly on the following hardware:
 - Raspberry Pi 1B
